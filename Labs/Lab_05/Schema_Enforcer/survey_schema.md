| Column Name | Required Data Type | Brief Description |
| :--- | :--- | :--- |
| `student_id` | `INT` | Unique identifier for the student. |
| `major` | `VARCHAR(100)` | Main area of study for the student. |
| `GPA` | `FLOAT` | Student's average grade on 4.0 scale. |
| `is_cs_major` | `BOOLEAN` | Indicates whether or not student is a CS major. |
| `credits_taken` | `FLOAT` | Number of credits student has completed. |
